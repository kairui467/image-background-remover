# Image Background Remover - 用户体系 & 定价策略设计

## 一、用户权限体系

### 1. 未登录用户
- **权限**：仅可查看首页和定价页面
- **操作**：
  - ❌ 无法上传图片
  - ❌ 无法使用背景移除功能
  - ✅ 可查看功能介绍
  - ✅ 可查看定价方案
  - ✅ 可点击"登录"按钮

### 2. 已登录用户（免费版）
- **初始额度**：3 次（注册赠送）
- **权限**：
  - ✅ 上传图片并处理
  - ✅ 下载处理结果
  - ✅ 查看个人中心
  - ✅ 查看使用记录
  - ✅ 升级到付费版
- **限制**：
  - 每月最多 3 次
  - 额度用完后需升级

### 3. 已登录用户（付费版）
- **权限**：同免费版 + 更多额度
- **限制**：根据订阅方案

### 用户转化路径

```

    ↓

    ↓
已登录用户（免费版）
    ↓

    ↓

    ↓

    ↓

    ↓
[继续使用] → [满意度高] → [续费]
```

### 关键转化点

#### 1. 首页转化
- **目标**：让未登录用户点击"开始使用"
- **策略**：
  - 突出"免费 3 次"

  - 简化登录流程

- **目标**：引导用户升级
- **策略**：
  - 弹窗提示"额度已用完"
  - 推荐"最受欢迎"的方案
  - 显示"节省 X 元"的优惠

#### 3. 定价页转化
- **目标**：让用户选择付费方案
- **策略**：
  - 突出"专业版"（中间方案）
  - 显示"最受欢迎"标签

## 六、数据库设计

### User 表扩展
```sql
CREATE TABLE User (
  id STRING PRIMARY KEY,
  email STRING UNIQUE,
  name STRING,
  image STRING,
  
  -- 额度系统
  credits INT DEFAULT 3,           -- 当前额度
  totalCreditsUsed INT DEFAULT 0,  -- 总使用额度
  
  -- 订阅系统
  planType STRING DEFAULT "FREE",  -- FREE, MONTHLY, YEARLY, CREDITS
  planExpiresAt DATETIME,          -- 订阅到期时间
  planStartedAt DATETIME,          -- 订阅开始时间
  
  -- 支付信息
  stripeCustomerId STRING,         -- Stripe 客户 ID（后期用）
  
  createdAt DATETIME DEFAULT NOW(),
  updatedAt DATETIME DEFAULT NOW()
);
```

### Subscription 表
```sql
CREATE TABLE Subscription (
  id STRING PRIMARY KEY,
  userId STRING FOREIGN KEY,
  planType STRING,                 -- MONTHLY, YEARLY, CREDITS
  planName STRING,                 -- 基础版, 专业版, 企业版
  price DECIMAL,
  creditsPerPeriod INT,
  startDate DATETIME,
  endDate DATETIME,
  status STRING,                   -- ACTIVE, CANCELLED, EXPIRED
  
  createdAt DATETIME DEFAULT NOW()
);
```

### Payment 表
```sql
CREATE TABLE Payment (
  id STRING PRIMARY KEY,
  userId STRING FOREIGN KEY,
  subscriptionId STRING FOREIGN KEY,
  amount DECIMAL,
  currency STRING DEFAULT "CNY",
  status STRING,                   -- PENDING, SUCCESS, FAILED
  paymentMethod STRING,            -- ALIPAY, WECHAT, PAYPAL
  transactionId STRING,
  
  createdAt DATETIME DEFAULT NOW()
);
```

### ImageRecord 表（扩展）
```sql
CREATE TABLE ImageRecord (
  id STRING PRIMARY KEY,
  userId STRING FOREIGN KEY,
  originalName STRING,
  status STRING DEFAULT "SUCCESS",
  cost INT DEFAULT 1,              -- 消耗的额度
  
  createdAt DATETIME DEFAULT NOW()
);
```

## 十、风险与对策

| 风险 | 对策 |
|------|------|
| 用户流失 | 提供优质服务、定期优惠 |
